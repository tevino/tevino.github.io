这是一个近况页面,我在这里写下我会告诉一年未见的朋友的东西。

更新于 2024-08-28 中国，苏州

有一次，一位朋友问我，作为独立开发者，我会给自己多长时间才重新回到朝九晚五的工作。我告诉他，我没有为自己设定时间限制。现在，一年多过去了，我们都同意——再也回不去了。

我从未像现在这样为自己而活。在此之前，我很少休假，有一种巨大的惯性让我保持每天工作的常规。我记得有一次，一位同事提醒我有即将过期的假期，我才意识到我还有去年结转的假期。几天后，就在假期即将过期之前，我第一次独自一人去另一个国家旅行。

在我独立开发之旅的开始，每一天都感觉像是度假，就像我辞职后总是有的感觉。现在，我以一种轻松的节奏构建我的项目。我还不确定这是好是坏，但这是我以前没有尝试过的生活方式。我希望我能继续这样生活下去。

更新于 2023-05-16 中国，苏州

当我即将加入微软时,我告诉我一些朋友,这将是我为他人工作生涯中的最后一个站点。

因为在那之后,我将遍历不同规模的公司实体:从零开始的创业公司,几十人的小公司,有和没有稳定现金流的中型公司,国内外行业巨头。

从微软辞职后,我有很多要分享的,但现在,简短的故事是,这个外国巨头与我想象的完全不同,在积极和消极方面都是如此。

现在我满足了好奇心,获得了经验,我知道这些只是片面的看法,但没有人会拥有全部。

现在对我来说是实现另一个目标的最佳时机:为自己工作。

我正在试验这种生活方式,看它是否更适合我。

关键是做一些我喜欢的事情,同时为他人创造价值,从而获得不必仅为赚钱而做令人厌恶的事情的特权。

更新于 2021-04-10 中国，苏州

加入微软已经一个星期了,到目前为止我对此很满意。

如我所料,新员工有许多必修培训,像无意识偏见这样的话题排在首位,我认为这非常重要,而在其他地方 rarely mentioned。

我已经有十年没把Windows作为主要桌面使用,甚至没有拥有任何运行它的设备,有趣的是,当我为工作得到运行Windows 10的工作站时,我非常擅长使用和自定义它,甚至我的一个同事对此感到惊讶“既然你是mac用户,为什么如此擅长使用Windows”,“猜我在第一台MacBook之前就是Windows的高级用户”我回答道,就在那时我意识到我惊人地顺利地应对了新的Windows工作站。

我必须说,在一个有几十个内核,几十GB内存,大SSD和管理良好的企业域的工作站上使用Windows 10体验非常愉快,用于在这个星球上最大的企业云工作。作为一个已经完全从Windows迁移到macOS十年的人,我对自己的说法感到震惊。

以下是我在微软观察到的一些好处:

  • 办公室大部分时间安静如图书馆
  • 人们互相称呼名字,没有花哨的头衔,也不会觉得被冒犯
  • 远程会议非常流畅,以至于
    • 你看不到拉扯的视频,方块和怪脸
    • 你听不到拉扯的声音和“你能说大声点吗?”
    • 你可以在屏幕上获得实时 Transcripts
    • 你不需要提高音量或对准麦克风说话给在线的人
  • 日历上的默认视图是“work week”,在工作日会屏蔽工作时间之外的小时数,整个周末也是

尽管我在第一周每天都加班(如果不想加班我不必这样做),但我可以看到工作和生活受到尊重,相比大多数国内企业,我认为外国企业承认员工确实有私人生活,这是他们必须共存的东西,所以他们选择尊重它,而国内企业选择忽略它,或者更糟的是不承认其存在,不允许员工在不让其他人觉得难过或气馁的情况下谈论它。

我不是说国内的就糟糕,仅仅因为它们是国内的,外国的存在时间更长,所以雇佣更成熟,我可以看到国内的正在学习,因为快乐的员工浸润在健康的文化中是大型企业的必需品,这只是需要时间。

人们倾向于认为大公司薪水越高,然而,微软以薪水低而闻名,在决定加入之前,我收到的offer是微软提供的数倍,猜我以后无论如何都会为我的任性付出代价。

我是一个喜欢生活在现代而不是过时陈旧的地方的人。我发现我不喜欢这座城市大部分较老的部分,好处是我确实在这座城市找到了一个像上海一样好的现代区域(甚至可能更好)。

职业

我在阿里巴巴集团的子公司之一担任高级软件工程师/经理,在中国非常大,但在该领域不是真正的顶级。尽管我不认为为别人工作是出路,但我想看看顶级技术公司是如何运作的。

健康

自新冠疫情暴发以来,我已经在家呆了两个多月,我最初试图保持肌肉训练,但很快就由于设备和蛋白质摄入的缺乏而放弃了。 现在我只是偶尔使用椭圆训练器来保持精力充沛。

学习

在意识到我所拥有的时间太少,无法做自己想做的事之前,我习惯学习对我新的、有趣的一切,这很多。 我的贪婪导致的问题不止这一个,为了让生活更轻松,我已经学会放下东西,放下我最不需要的东西。 我还不是很确定这是不是正确的做法。
